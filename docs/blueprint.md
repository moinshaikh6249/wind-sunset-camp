# **App Name**: Sunset Camp

## Core Features:

- Homepage Hero Banner: Displays camping images and key call-to-action buttons.
- About Us Page: Provides details about the organization's mission and team.
- Gallery Page: Loads and displays camping images in a grid layout.
- Upcoming Camps Listing: Fetches and displays upcoming camp details.
- Booking Form Completion Suggester: AI-powered tool to help users auto-complete sections of the booking form to reduce user effort
- Contact Page: Displays contact details and a message submission form.

## Style Guidelines:

- Primary color: Forest green (#228B22) to evoke nature and the outdoors.
- Background color: Light beige (#F5F5DC) for a natural, earthy feel.
- Accent color: Burnt orange (#CC6633) to highlight key actions.
- Body font: 'PT Sans' sans-serif for a modern and clean readability.
- Headline font: 'Playfair' serif to provide an elegant header. (Note: currently only Google Fonts are supported.)
- Use line icons to match modern design aesthetic.
- Ensure a responsive layout that adapts to various screen sizes.
